# Jeu Mastermind en Java

## Description
Jeu classique de Mastermind implémenté en Java. L'objectif est de deviner une combinaison de couleurs en moins de 12 tentatives. À chaque tentative, le jeu indique combien de couleurs sont bien placées et combien sont présentes mais mal placées.

## Installation
- Téléchargez ou clonez le projet.
- Assurez-vous d'avoir Java 8 ou une version supérieure installée sur votre machine.
- Compilez et exécutez le projet avec votre IDE préféré (par exemple, NetBeans, Eclipse).

## Usage
Le jeu se joue directement dans la console, où l'utilisateur doit entrer ses propositions et obtenir les résultats après chaque tentative.
